# 风格提取 Skill

分析你的写作风格，生成风格画像，帮你保持一致的文风写新内容。

## 触发条件

✅ **USE this skill when:**

- 用户说「分析我的写作风格」「提取风格」
- 用户说「用我的口吻写」「像我一样写」
- 用户说「保持我的风格」「文风一致」
- 用户想要了解自己的写作特点

❌ **DON'T use this skill when:**

- 用户想要改变写作风格
- 用户想要学习他人的写作风格

## 工作流程

### 风格分析模式

1. **采样文档** - 获取用户近期文档（建议 5-10 篇）
2. **多维分析** - 分析词汇、句式、结构、语气等维度
3. **生成画像** - 输出结构化的风格画像
4. **保存画像** - 存储为语雀文档供后续使用

### 风格应用模式

1. **读取画像** - 获取已保存的风格画像
2. **接收需求** - 用户提供写作任务
3. **风格写作** - 按画像风格生成内容
4. **保存文档** - 创建新文档到语雀

## 风格画像维度

```markdown
# ✍️ 写作风格画像

## 📊 基础特征

- **平均句长**: 18 字
- **段落长度**: 3-5 句
- **标点偏好**: 喜用破折号、省略号
- **数字风格**: 阿拉伯数字为主

## 🎨 语言风格

- **正式度**: ⭐⭐⭐☆☆ (偏口语化)
- **情感色彩**: 积极、鼓励性
- **幽默程度**: 适度幽默
- **专业术语**: 中等使用

## 📐 结构偏好

- **标题风格**: Emoji + 关键词
- **开头方式**: 直接切入主题
- **段落组织**: 总分总结构
- **结尾风格**: 行动号召或金句

## 💬 特色表达

- 常用口头禅: 「简单来说」「换句话说」
- 喜欢使用: 比喻、类比
- 标志性结构: 问答式引入

## 🏷️ 代表作品

1. [文档1](link)
2. [文档2](link)
```

## MCP 调用

```bash
# 获取用户知识库列表
mcporter call yuque.yuque_list_repos

# 获取知识库文档列表
mcporter call yuque.yuque_list_docs repo_id="blog"

# 获取文档内容用于分析
mcporter call yuque.yuque_get_doc repo_id="blog" doc_id="post-1"

# 保存风格画像
mcporter call yuque.yuque_create_doc \
  repo_id="system" \
  title="✍️ 我的写作风格画像" \
  body="画像内容"
```

## 使用示例

```
用户: 分析我的写作风格，然后用我的风格写一篇关于 AI 的文章

助手:
1. 获取用户近 10 篇文档
2. 分析写作特征
3. 生成风格画像
4. 按画像风格撰写 AI 文章
5. 保存到语雀
```
